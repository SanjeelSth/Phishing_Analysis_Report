# Phishing Email Analysis

## Overview
This project involves the in-depth analysis of a real-world phishing email sample. The goal was to identify the tactics, techniques, and indicators of compromise (IOCs) used by attackers to deceive victims — and to document findings in a structured report.

## Objectives
- Identify phishing indicators within a real email sample
- Analyse the social engineering techniques used by the attacker
- Document findings clearly for a security audience
- Build practical threat analysis skills relevant to SOC and cybersecurity roles

## What Was Analysed
- Sender information and spoofed addresses
- Email subject line and urgency tactics
- Malicious or suspicious links embedded in the email
- Social engineering techniques used to manipulate the recipient
- Overall attack intent and likely target profile

## Key Findings
The analysis identified several red flags commonly used in phishing campaigns, including sender spoofing, deceptive URLs, and urgency-based manipulation tactics designed to bypass critical thinking.

## Skills Demonstrated
- Threat analysis and documentation
- Understanding of phishing attack vectors
- Report writing for a cybersecurity context
- Knowledge of social engineering techniques
